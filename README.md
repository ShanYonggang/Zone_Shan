- 本网站是一个个人博客网站，主要分享博主的编程学习心得
- 网站主要使用 django + mysql + nginx + uwsgi搭建，项目部署在阿里云服务器上
- 网站用到的很多技术其实并非必须，但是我做这个网站的目的是为了把所学的东西都运用起来，所以会尽可能使用更多的技术去扩展网站的功能
- 博客源码已全部放置在github上
- 网站功能不断更新，具体可以查看网站发展状况
